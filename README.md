# ExamPP
专为中国高中生设计，根据3+1+2选科政策定制学习计划，提供全科复习资料、进度追踪与目标管理，助力高效备考。包含AI助手等功能，保存在本地更安全。包含小语种。           （仅支持中文版）                      （由于部分原因，不支持导出功能）


Designed specifically for Chinese high school students, this program customizes study plans based on the 3+1+2 subject selection policy, providing comprehensive review materials, progress tracking, and goal management to help with efficient exam preparation. It includes features like an AI assistant and is stored locally for added security. It also covers minor languages. (Only available in Chinese version) (Due to certain reasons, export function is not supported)
